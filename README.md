# E Commerce Homepage Wireframe – DecodeLabs Project 2

## Overview

This project was created as part of the DecodeLabs UI/UX Internship Program under **Project 2: The Blueprint (Wireframing)**.

The objective of this project was to redesign the structure of an E commerce landing page using low fidelity wireframes while focusing on:

- User Flow
- Information Hierarchy
- Clear Navigation
- Improved Checkout Experience
- Reduced Visual Clutter

This project focuses on usability and structure rather than visual aesthetics.

---

# Problem Statement

The scenario presented an E commerce brand losing customers during the checkout process because the interface was cluttered and confusing.

The goal was to create a clean and logical homepage structure that guides users naturally through the shopping experience.

---

# My Approach

While working on this project, I focused on simplicity, clarity, and usability.

Instead of using colors, images, or visual effects, I concentrated on:

- Organizing content properly
- Creating a clean layout
- Making important actions easy to notice
- Improving readability and scanning
- Designing a user friendly flow from browsing to checkout

I used spacing, alignment, hierarchy, and clear Call to Action buttons to improve the overall user experience.

---

# Features Included

- Header Navigation
- Search Bar
- Hero Section with CTA
- Product Categories
- Featured Products Section
- Add to Cart Buttons
- Benefits Section
- Newsletter Subscription
- Structured Footer

---

# Design Decisions

Some of the design decisions made during this project include:

- Using a simple grid layout to reduce clutter
- Prioritizing the hero section to guide users immediately
- Keeping CTA buttons visible and accessible
- Using consistent spacing for better readability
- Organizing products into structured cards for easier scanning
